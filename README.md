# OCR CV Project

This project implements Optical Character Recognition (OCR) using Computer Vision techniques in Python.
It is developed and executed using Google Colab.

## Features
- Image preprocessing using OpenCV
- Text extraction using Tesseract OCR
- Noise removal and thresholding
- Works on scanned images and document images

## Tech Stack
- Python
- OpenCV
- Tesseract OCR
- NumPy
- Pillow
- Matplotlib
- Google Colab

## How to Run (Google Colab)
1. Open the notebook in Google Colab
2. Run all cells in sequence
3. Upload or provide image paths when prompted
